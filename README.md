# Inventory Management System (TP0_JSP)

Une application web Java (Jakarta EE) permettant de gérer un inventaire de produits et une liste d'utilisateurs. Ce projet a été réalisé dans le cadre d'un travail pratique (TP) pour apprendre à lier des pages web dynamiques (JSP) avec une base de données relationnelle via un ORM.

## Technologies utilisées

* **Langage :** Java 17
* **Frontend :** JSP (JavaServer Pages), HTML, CSS
* **Backend :** Servlets (Jakarta EE)
* **Serveur Web :** Apache Tomcat 10.1.19
* **Base de données :** MySQL
* **ORM :** Hibernate 6.4
* **Outil de build :** Maven (avec le plugin Cargo pour Tomcat)

## Fonctionnalités principales

* **Gestion des produits :**
  * Afficher la liste complète des produits.
  * Ajouter un nouveau produit avec ses détails (Nom, SKU, Prix, Quantité en stock, Description).
* **Gestion des utilisateurs :**
  * Afficher la liste des utilisateurs enregistrés.
  * Ajouter un nouvel utilisateur (Nom, Prénom, Email, Mot de passe).

## Prérequis

Pour faire tourner ce projet sur votre machine locale, vous aurez besoin de :
* [Java JDK 17](https://www.oracle.com/java/technologies/javase/jdk17-archive-downloads.html) ou supérieur.
* [Maven](https://maven.apache.org/) installé.
* Un serveur [MySQL](https://dev.mysql.com/downloads/mysql/) en cours d'exécution.
